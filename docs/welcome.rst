Welcome
=======

**sda-mc-simulator** is a Monte Carlo simulator for sequential decision problems using pluggable policies, transition models, and bootstrap exogenous samplers.

This library provides a flexible framework for simulating and analyzing sequential decision-making under uncertainty. Whether you're studying inventory management, supply chain optimization, or other dynamic optimization problems, this simulator lets you:

- Define custom policies to guide decision-making
- Use pluggable transition models to simulate system dynamics
- Bootstrap exogenous noise and external factors
- Extract and analyze metrics from simulation runs
